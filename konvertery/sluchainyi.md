# Add

Конвертер в параметре "Глубина" добавляет три подпараметра:

* "Связь"
* "Добавление"
* "Скаляр"

Конвертер "Add" можно использовать с параметрами следующего типа: угол, цвет, градиент, целое число, реальное число, время и вектор.
